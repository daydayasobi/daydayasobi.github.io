TEngine框架学习（1）摘要

DayDayAsobi's Blog
2025-05-27 (Updated: 2025-05-27)
TEngine, Unity

1. 前言

​ 在博主接触TEngine框架之前，最先看到的是GameFramewrok框架（后面简称GF框架），不得不说，GF的代码写的堪称优雅，每次去看源码都有新的收获。但是呢GF框架也存在两个问题，一个是教程太少，官方文档手册本来就是六脉残卷了，民间的个人教程质量也是参差不齐，时至今日能找到的有用教程也不多。还有一个问题是GF框架原工程已经没有在维护了，有一些功能有点过时了，可以用更好的插件代替。

​ 但是另一方面，GF框架有着非常好功能架构，有很多大佬在GF框架的基础上进行了优化和重构，引入了很多优秀的插件，比如YooAsset和HybridCLR等等。这些框架中也就包括这个文章的主题TEngine框架（后面简称TE），下文是TE的简介。

TEngine是一个简单(新手友好开箱即用)且强大的Unity框架全平台解决方案,对于需要一套上手快、文档清晰、高性能且可拓展性极强的商业级解决方案的开发者或者团队来说是一个很好的选择。

​ 让我自己评价TE的话，TE充分利用了GF框架的高可扩展性，重构后更加的高内聚低耦合。并且引入了很多久经考验的商业级插件，能够轻松实现商业化热更、商业化资源管理、商业化UI开发。如果你已经非常熟悉Luban、YooAsset等插件，那么这一切只需要下载工程开箱即用，真正如作者所说*“5分钟即可上手整套开发流程”。不仅如此，在这些基础上还有着*官方文档清晰（你别管文档全不全，就说清晰不清晰吧）、bug修复迅速、社区活跃的优势，总之是一款值得去研究和使用的Unity框架。

​ 话说回来，其实TE框架的demo分支里已经包含了很完善的一个打飞机Demo，但是本着学习的心态，博主打算效仿花桑大佬，复刻一个基于TE框架的塔防Demo，同时记录一下自己在使用TE框架心得和遇到的坑，希望能给后来人一些经验和启发。

2. 相关的插件和项目

  • GameFramework: Game Framework 是一个基于 Unity 引擎的游戏框架，主要对游戏开发过程中常用模块进行了封装，很大程度地规范开发过程、加快开发速度并保证产品质量。

  • TEngine: 前面已经介绍过了。

  • TowerDefense-GameFramework-Demo: 这是花桑大佬基于

  • Tower Defense Template: 这是一款官方的塔防模板资源。

  • YooAsset: YooAsset是一套商业级经历百万DAU游戏验证的资源管理系统。

  • HybridCLR: HybridCLR是一个特性完整、零成本、高性能、低内存的近乎完美的Unity全平台原生c#热更新解决方案。

  • Luban: luban是一个强大、易用、优雅、稳定的游戏配置解决方案。它设计目标为满足从小型到超大型游戏项目的简单到复杂的游戏配置工作流需求。
